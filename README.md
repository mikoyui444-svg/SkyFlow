# SkyFlow
Skyflow is a physics-based platformer built with C#. Players must jump between platforms, collect coins, avoid falling, and survive as long as possible. The game features a scoring system, player lives, checkpoint and respawn mechanics, bonus platforms, collectible rewards, an in-game economy, shop upgrades, multiple difficulty levels, audio controls, and progression systems. Special attention was given to gameplay balance, platform generation, probability-based bonus spawning, and player progression.

# SkyFlow

The objective is to jump across platforms, collect coins, achieve the highest score possible, and survive increasingly challenging gameplay.

## Features

* Physics-based movement and jumping
* Dynamic platform generation
* Score tracking system
* Coin collection system
* Shop and in-game economy
* Player lives and respawn mechanics
* Checkpoints
* Bonus platforms and rewards
* Multiple difficulty levels
* Audio and gameplay settings
* Progression system

## Technologies

* C#
* Unity

## Project Goals

This project was created to explore game mechanics, probability systems, progression design, and software development principles through an interactive application.
